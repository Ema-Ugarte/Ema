## Hola, soy Ema, y este es mi primer proyecto.

Para aclararlo desde el principio: este proyecto fue revisado más de dos veces porque cometí errores. Por eso, si ves celdas en verde, amarillo o rojo, es porque alguien más reviso mi revisó mi trabajo.
Solo quería dejarlo claro, porque prefiero ser honesto.

En este reto enfrenté muchos desafíos, porque no sabía que hacía cada método. Tenía medio. No entendía bien cosas como el método split, replace, como calcular el máximo o el míninmo, o cómo convertir un string a un elemento en una lista. Al ver muchas variables o al pensar que tenía que hacer algo grande, me detenía. Dudaba de mí.

Pero incluso con ese miedo, lo hice. Y aquí está. Este proyecto representa mi inicio, mis errores y mi decisión de seguir adelante. No sé explicarlo todo perfeccto todavía, pero con el tiempo... eso también va a cambiar.
